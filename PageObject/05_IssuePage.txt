# Positive Test Cases


Create Issue
    Navigate To Projects   Projects_Url
    Open Project   Project_Name
    Navigate To Issues Page
    Click New Item Button
    Input Issue Title   Issue_Title
    Input Issue Description (Optional)   Issue_Description
    Click Create Issue Button
    Verify Page Should Contain  "Issue created. View details."


Update Name Of Issue
    Navigate To Project Page  Projects_Url
    Open Project   Project_Name
    Navigate To Issues Page
    Click on Issue Title   Issue_Title
    Click Edit Button
    Update Issue Title   Updated_Issue_Title
    Click Save Changes
    Verify Page Should Display  Updated_Issue_Title


Create Second Issue With Previously Updated Name
    Navigate To Project Page  Projects_Url
    Open Project   Project_Name
    Navigate To Issues Page
    Click New Item Button
    Input Issue Title    Updated_Issue_Title
    Input Issue Description (Optional)  Issue_Description
    Click Create Issue Button
    Verify Page Should Contain  "Issue created. View details."


Delete First Issue
    Navigate To Project Page  Projects_Url
    Open Project   Project_Name
    Navigate To Issues Page
    Select Issue   Issue_Title
    Click Three Dots in The Upper Right Corner 
    Click Delete Issue Option
    Confirm Deletion
    Verify Issue Is No Longer Visible In Issue List


# Negative Test Cases

Creating Issue with Empty Issue Name
	Navigate To Projects   Projects_Url
    Open Project   Project_Name
    Navigate To Issues Page
    Click New Item Button
    Do NOT Input Issue Title   Empty
    Input Issue Description (Optional)   Issue_Description
    Click Create Issue Button
    Verify Page Should Contain  "A title is required" (focused title field)
