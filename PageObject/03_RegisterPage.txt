
### **User Registration TestSuite (Collection of Test Cases) Overview**

#******************** Settings ************************

*** Settings ***

Documentation: 

Testing the registration page.
Test scenarios include positive tests for valid data entry, and negative tests for invalid inputs 
(for example, empty fields, invalid email, invalid or weak password). The goal is to make sure 
that account creation works correctly and correct error messages are displayed based on context.


Pre Condition - Open browser
Post Condition - Closed all browsers
Tags - Register, TS-2-Register
